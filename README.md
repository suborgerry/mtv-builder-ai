# MTV Builder AI

A minimal frontend project for building an AI-driven page builder workflow: page structure, section rules, content constraints, and render specifications.

At the moment, the repository is a lightweight Vite starter with TypeScript, Tailwind CSS v4, and a collection of JSON references that can serve as a foundation for page generation, validation, and assembly.

## Stack

- TypeScript
- Vite
- Tailwind CSS v4

## What's Inside

- `src/` - application source code
- `public/` - static assets
- `json_references/` - reference JSON schemas and pipeline artifacts
- `to_do.md` - current notes for upcoming tasks

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Status

The project is still at an early stage: the base infrastructure is in place, while the core builder logic is still being shaped.
